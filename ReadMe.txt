1. 多Que es una clase "sealed"?
	Un clase ""sealed"/sellada se utiliza cuando queremos restringir el ambito de herencia de la misma,
	es decir, que no se pueda heredar.

2. 多Cuales son los operadores de una expresion relacional?
	Los operadores de las expresiones racionales son los siguientes:
		< menor que
		> mayor que
		<= menor o igual que
		>= mayor o igual que
		== igual que
		!= distinto que
